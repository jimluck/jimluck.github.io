Website Evaluations

Good Website

This is a description of the good website which follows good design principles.

  • Contrast: The website has high contrast between text and background ensuring readability.
  • Repetition: Design elements are consistently used throughout the pages creating a cohesive feel.
  • Alignment: The content is well aligned which guides the user's eyes in a logical flow.
  • Proximity: Related items are grouped together, making the site easy to navigate.

Bad Website

This is a description of the bad website which lacks fundamental design principles.

  • Contrast: Poor contrast makes it difficult to read text against backgrounds.
  • Repetition: There is a lack of consistency in the use of design elements.
  • Alignment: Content is randomly placed, creating a confusing layout.
  • Proximity: Items are scattered, making the site hard to use.
