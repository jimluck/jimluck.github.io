Website Evaluations

Good Website: UNC Charlotte

>

The UNCC webpage seems to be designed with the CRAP principles in mind:

  • Contrast: The webpage exhibits a strong contrast with a clear and legible typography set against the background, which is crucial for readability. The use of white text on a semi-opaque overlay over a dynamic image ensures that the text stands out. Moreover, the green color used for the navigation bar and buttons is consistent with the institution's branding and contrasts well with the white text.
  • Repetition: The design elements such as color scheme, typography, and button style are consistent across the webpage, creating a sense of unity and branding. The green color is repeated in key areas, which ties the various parts of the site together visually. This repetition aids in brand recognition and provides a cohesive user experience.
  • Alignment: Content on the page follows a clear grid structure, with text, buttons, and other elements aligned both vertically and horizontally. This structured alignment helps create a balanced and easily navigable page, guiding the user's eye through the content in a logical sequence.
  • Proximity: Related items on the webpage are grouped together effectively. For example, navigation links are placed close to each other at the top of the page, while calls to action such as "About Us," "Apply Now," and "Visit Our Campus" are grouped in a dedicated area, making it intuitive for the user to find related options.

Bad Website: Monkzone

This website is evaluated based on the CRAP design principles as follows:

  • Contrast: The website appears to use electric blue text on a black background, which might create a glare effect and reduce readability. The use of contrasting colors is a critical element of design, as it makes the content stand out and be legible. In this case, the high contrast between the blue text and black background may be straining for the eyes, especially given the small text size.
  • Repetition: At first glance, there does not appear to be a consistent theme or repetition of design elements that unify the site. Repetition in design helps to create consistency and a sense of cohesion across a website, which seems to be lacking here.
  • Alignment: The text and other elements on the page appear to be aligned in a somewhat arbitrary manner, lacking a clear and consistent structure. Proper alignment is crucial for creating a polished, professional look and for guiding the user's eye in a logical flow across the page.
  • Proximity: Items that are related to each other should be grouped together to help users process the information. The page seems to have a scattered layout, which might confuse users and make the navigation of the site difficult.
